计算公式说明
方法一：基于载荷
F₀₂ = (G₀ × r − m₀) × g / 2
F_w2 = (G_w − m₀) × g / 2
F_k2 = 0.5(F₀₂ + F_w2)
λ = F_w2 / F₀₂
C_a2/C_m2 = (2λ − 2)/(λ + 3)
f_c2 = (4.98 / n₂)² [cm]
注：f_c2单位为cm，需转换为mm使用
方法二：基于质量
M₀₂ = (G₀ × r − m₀) / 2
M_w2 = (G_w − m₀) / 2
M_k2 = 0.5(M₀₂ + M_w2)
λ = M_w2 / M₀₂
C_a2/C_m2 = (2λ − 2)/(λ + 3)
f_c2 = g/(2π·n₂)² × 1000 [mm]
